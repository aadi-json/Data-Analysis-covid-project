This project focuses on exploratory data analysis and business intelligence visualization of global COVID-19 data using Python, Pandas, Matplotlib, and Seaborn.
The objective is to transform raw pandemic data into meaningful insights that can support policy decisions, healthcare planning, and strategic analysis.

The analysis covers country-level and WHO region-level metrics, including confirmed cases, deaths, recoveries, active cases, mortality risk, and recovery rates. A comprehensive multi-visual dashboard is created to present trends, relationships, and distributions in a clear and interpretable manner.
